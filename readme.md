
# 🌍TWN Website

##  💻Run Locally

Install Node.js ,
Update Config file

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
 npm install 
```

Start the server

```bash
  npm start
```
